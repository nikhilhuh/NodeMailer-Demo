üíå STEP 1: GENERATE GMAIL APP PASSWORD (GMAIL_USER and GMAIL_PASS)
------------------------------------------------------------------

‚ö†Ô∏è Gmail no longer allows less secure apps to access your account using your regular password.  
You must use an **App Password**.

Follow these steps to generate it:

(1) Enable 2-Step Verification:
    ‚û§ Visit: https://myaccount.google.com/security  
    ‚û§ Under ‚ÄúSigning in to Google‚Äù, turn ON 2-Step Verification

(2) Create an App Password:
    ‚û§ Go to: https://myaccount.google.com/apppasswords  
    ‚û§ Select **Mail** as the app  
    ‚û§ Choose **Other (Custom name)** and name it ‚ÄúNodemailer‚Äù  
    ‚û§ Click **Generate**

(3) Copy the 16-character password (e.g., abcd efgh ijkl mnop)  
    ‚û§ Store it securely

(4) Use your Gmail address as:      GMAIL_USER  
    Use the 16-char App Password as: GMAIL_PASS

---

üì¶ STEP 2: CREATE `.env` FILE
------------------------------
In your project root, create a `.env` file and add:

    GMAIL_USER=yourgmail@gmail.com  
    GMAIL_PASS=your16charAppPassword  

(‚úî Make sure `.env` is listed in `.gitignore` to avoid leaking credentials)

---

üì¨ STEP 3: CREATE `sendMail.js` FILE
-------------------------------------
Use this sample code (if not already created):

const nodemailer = require('nodemailer');
require('dotenv').config();

const sendMail = async (to, subject, text) => {
  try {
    const transporter = nodemailer.createTransport({
      service: 'gmail',
      auth: {
        user: process.env.GMAIL_USER,
        pass: process.env.GMAIL_PASS,
      },
    });

    const mailOptions = {
      from: process.env.GMAIL_USER,
      to,
      subject,
      text,
    };

    const result = await transporter.sendMail(mailOptions);
    console.log("‚úÖ Email sent:", result.response);
  } catch (err) {
    console.error("‚ùå Error sending email:", err);
  }
};

module.exports = sendMail;
